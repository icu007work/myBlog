## 高通方案编译及刷机流程(以45S为例)

### 1. 编译

```bash
source build/envsetup.sh SQ45S && setenv && make update-api && make dist -j16 2>&1 |tee build.log
```

### 2. 生成刷机包路径

一般都在source/out/dist 以及 source/out/target

### 3. 刷机流程（sideload）

1. 进入recovery模式

```bash
adb reboot recovery
```

2. 音量键：+-+-++--，进入recovery界面后选择：Apply update from ADB
3. 输入：

```bash
adb sideload 刷机包
```

4. 等待刷机完成后重启设备。
